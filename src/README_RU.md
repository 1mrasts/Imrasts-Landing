## Imrasts-Landing

Этот репозиторий — реализация моего персонального сайта-визитки, собранного по макету, который я сам делал в Figma.

Идея была простая — сделать чистый, аккуратный лендинг без лишнего шума, но с нормальной архитектурой и живыми UI-элементами.

## Стек технологий

### React + Vite

- **React** — база проекта. Вся логика и интерфейс построены на React с упором на чистую структуру и читаемость кода (без лишней каши и хаоса).
- **Vite** — быстрый билд и комфортная разработка без долгих ожиданий.

### Mobile-first подход

Вёрстка изначально строилась от мобильной версии, дальше уже масштабировалась под более крупные экраны.

## Функциональность и особенности

### UI / UX фишки

- Анимация появления компонентов при загрузке страницы
- Полная адаптивность под большинство устройств
- Локализация: поддержка русского и английского языков

---

Проект делался как практическая работа над лендингом + прокачка архитектуры и аккуратной вёрстки без перегруза лишней логикой. Так же упор был на анимации и адаптив с mobile-first подходом
